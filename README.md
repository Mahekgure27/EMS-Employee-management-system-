# Employee Management System (EMS)

## 📌 About The Project

Employee Management System (EMS) is a web-based application developed using React and Vite.
The project helps admins manage employee tasks efficiently by assigning tasks, monitoring progress, and tracking task status.

This project was created for learning full-stack development concepts and improving frontend development skills.

---

## 🚀 Features

* Admin Login
* Employee Login
* Task Assignment System
* Task Status Management
* Completed Task Tracking
* Failed Task Tracking
* Responsive UI
* Dashboard for Admin and Employees

---

## 🛠️ Technologies Used

* React.js
* Vite
* JavaScript
* Tailwind CSS
* HTML5
* CSS3

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Mahekgure27/EMS-Employee-management-system-.git
```

Move into project folder:

```bash
cd EMS-Employee-management-system-
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
